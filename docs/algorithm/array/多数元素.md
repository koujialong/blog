# 多数元素 LC169

## 题目
给定一个大小为 n 的数组 nums ，返回其中的多数元素。多数元素是指在数组中出现次数 大于 ⌊ n/2 ⌋ 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。
* 输入：nums = [2,2,1,1,1,2,2]
* 输出：2

# 题解
```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var majorityElement = function (nums) {
  const map = {};
  let max = 0;
  let res;
  nums.forEach((num) => {
    if (map[num]) {
      map[num]++;
    } else {
      map[num] = 1;
    }
    if (map[num] > max) {
      max = map[num];
      res = num;
    }
  });
  return res;
};
```
