# 翻转二叉树 LC226

## 题目
给你一棵二叉树的根节点 root ，翻转这棵二叉树，并返回其根节点

![img](/img/tree226.png)

* 输入：root = [4,2,7,1,3,6,9]
* 输出：[4,7,2,9,6,3,1]

## 题解
```javascript
/**
 * @param {TreeNode} root
 * @return {TreeNode}
 */
var invertTree = function (root) {
  if (root === null) {
    return null;
  }
  const left = invertTree(root.left);
  const right = invertTree(root.right);
  root.left = right;
  root.right = left;
  return root
};
```
